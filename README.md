# quicknote
Simple application to take quick notes from notifications, without losing focus of current running app

you can download the zip file to get the Android studio project, for quick install simply download the apk file in files and install it.

# Install Notes

please enable notifications permission and draw over any apps permission , both need to be given, app currently doesn't ask for permission still you need to give permission both for it to work. also there is 
# no saving of notes
so for now you need to take screenshots to persist. use case is primarily for when you are playing any game or seeing movies, serials etc on phone and don't want primary app to lose focus , i.e your game or movie etc to pause, and want to take quick note.
